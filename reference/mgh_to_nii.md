# Convert 'FreeSurfer' `'mgh'` to 'Nifti'

Convert 'FreeSurfer' `'mgh'` to 'Nifti'

## Usage

``` r
mgh_to_nii(from, to)
```

## Arguments

- from:

  path to 'FreeSurfer' `'mgh'` or `'mgz'` file

- to:

  path to 'Nifti' file, must ends with `'nii'` or `'nii.gz'`

## Value

Nothing; the file will be created to path specified by `to`
