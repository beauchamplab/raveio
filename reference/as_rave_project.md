# Convert character to [`RAVEProject`](RAVEProject.md) instance

Convert character to [`RAVEProject`](RAVEProject.md) instance

## Usage

``` r
as_rave_project(project, ...)
```

## Arguments

- project:

  character project name

- ...:

  passed to other methods

## Value

A [`RAVEProject`](RAVEProject.md) instance
