# Get 'Neurosynth' website address using 'MNI152' coordinates

Get 'Neurosynth' website address using 'MNI152' coordinates

## Usage

``` r
url_neurosynth(x, y, z)
```

## Arguments

- x, y, z:

  numerical values: the right-anterior-superior 'RAS' coordinates in
  `'MNI152'` space

## Value

'Neurosynth' website address
